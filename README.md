# student-performance-EDA
# Student Performance Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on a student performance dataset to identify the key factors that influence academic results.
The analysis includes data exploration, statistical summaries, and visualizations to understand patterns and relationships within the dataset.

The goal of this project is to discover insights that can help educators and institutions understand student performance and support data-driven decision making.

---

## 🎯 Objective

The main objectives of this project are:

* To explore and understand the structure of the student performance dataset.
* To perform **data cleaning and preprocessing**.
* To generate **descriptive statistics** for summarizing the data.
* To create **visualizations** that help in understanding distributions and relationships between variables.
* To identify factors that influence student academic performance.

---

## 📂 Dataset

The dataset used in this project is obtained from the **UCI Machine Learning Repository**.

It contains information about students including:

* Demographic details
* Study habits
* Parental education
* School related attributes
* Student grades

Dataset link:
https://archive.ics.uci.edu/ml/datasets/Student+Performance

The dataset includes **395 student records and 33 attributes**.

## 📊 Exploratory Data Analysis (EDA)

The following steps were performed during the analysis:

### 1️⃣ Data Exploration

* Viewing the first few rows of the dataset
* Checking dataset shape and column names
* Understanding data types and structure

### 2️⃣ Data Cleaning

* Checking for missing values
* Verifying dataset consistency

### 3️⃣ Descriptive Statistics

* Calculating mean, median, minimum, maximum, and standard deviation
* Understanding the distribution of numeric features

### 4️⃣ Data Visualization

Several visualizations were created to analyze the dataset:

* Distribution of final grades
* Gender vs student performance
* Study time vs final grades
* Absences vs academic performance
* Correlation heatmap for numeric variables

These visualizations help identify patterns and relationships between different variables.

---

## 🔍 Key Insights

Some important findings from the analysis include:

* Previous grades (**G1 and G2**) strongly influence the final grade (**G3**).
* Students who spend more time studying generally achieve better academic performance.
* Higher student absences are associated with lower final grades.
* Parental education shows a moderate relationship with student performance.
* The dataset does not contain missing values, making it suitable for analysis.
